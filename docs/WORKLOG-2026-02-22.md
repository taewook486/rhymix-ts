## OAuth 테스트 참고사항
Google/GitHub OAuth 로그인은 Supabase Dashboard 설정이 필요:
1. Supabase Dashboard → Authentication → Providers
2. Google/GitHub provider 활성화
3. Callback URL: `https://your-domain.com/auth/callback`
4. Vercel 배포 후 테스트 권장
